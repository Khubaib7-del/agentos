# thruline

Companion layer for AI coding agents — review queue, decision locks, session
snapshots, and cross-agent project memory for Claude Code, Cursor, Copilot,
Codex, and Gemini CLI.

```
npm install -g thruline
thruline init
```

This package downloads the prebuilt `thruline` binary for your platform from
[GitHub Releases](https://github.com/Khubaib7-del/thruline/releases) and
verifies its SHA-256 against checksums pinned inside this package before
anything runs. Binaries are built by CI with Sigstore provenance.

Full documentation: https://github.com/Khubaib7-del/thruline
