Three commands, and the amnesia is gone

No cloud, no account, no telemetry. Everything lives in plain files inside your repo.

01

Remember

thruline decide "DB: PostgreSQL" --lock — decisions ride along with every prompt, in every session, forever. Agents flag conflicts instead of silently deviating.

02

Steer without interrupting

thruline note "add debounce" while the agent works. When it finishes, your notes arrive as review comments — automatically in Claude Code, on request everywhere else.

03

Never start from zero

thruline snapshot bottles the session; thruline restore pours it into any future session — or a completely different agent.

Built like infrastructure, not a toy

Written in Rust. Single binary. 29 tests, three-platform CI, and a public threat model.

Decision locks

Locked choices are injected into every prompt. If an agent proposes MongoDB after you locked PostgreSQL, it gets flagged — before code changes.

Review queue

Ideas strike while the agent is busy. Queue them from any terminal; they're delivered like code-review comments, never as interruptions.

Cross-agent memory

One memory, rendered into AGENTS.md and served over MCP — readable by every major agent, so switching tools doesn't reset your project's story.

Context health

Live context %, estimated prompts left, and usage-window reset time in your Claude Code statusline. When it can't know, it says nothing — never a made-up number.

Secret redaction

Eleven credential formats — API keys, tokens, JWTs, connection strings — are scrubbed before anything touches disk. A pasted key can't leak through your decision log.

Tamper quarantine

Memory edited outside Thruline (a malicious git pull, a rogue script) is quarantined from agents until you review and approve it. Fingerprints live outside the repo.

Security is the product

  • Local-first: no cloud, no account, no telemetry — ever
  • Binaries built and signed by CI only, with Sigstore provenance
  • npm installs verify SHA-256 against pinned checksums
  • Fail-open hooks: a Thruline bug can never block your agent
  • No trust-approval slash command — humans approve memory, not agents
  • Public threat model with nine documented findings
